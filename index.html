Proyecto

Sistema web para centro médico: agendamiento, dashboard y comunicación post-consulta

Descripción:

Este es un sistema web desarrollado para el Centro Médico FamySALUD con el objetivo de optimizar la gestión de citas, mejorar la toma de decisiones administrativas y fortalecer la comunicación con los pacientes. Cuenta con tres módulos principales:

  • Agendamiento de citas: Permite a los usuarios reservar consultas de forma fácil y rápida.
  • Dashboard administrativo: Proporciona estadísticas clave para la gestión eficiente.
  • Comunicación post-consulta: Ofrece encuestas de satisfacción y recordatorios de citas.

Tecnologías usadas:

  • PHP
  • JavaScript
  • AJAX
  • HTML5, CSS3
  • MySQL
  • Chart.js
  • Twilio
  • XAMPP
